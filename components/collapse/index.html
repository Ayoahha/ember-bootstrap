Collapse

Easily hide and reveal additional content.

This page shows some example usage of the <BsCollapse> component. For the full documentation please refer to the API Docs. You might also want to consult the official Bootstrap Docs for general information.

This is collapsible content

<BsCollapse @collapsed={{this.collapsed}}>
  <div class="well">
    <p>This is collapsible content</p>
  </div>
</BsCollapse>
